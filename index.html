About

This is an intentionally lightweight RSS reader: it runs entirely in your browser with no dedicated servers, so there’s no lock-in, no server costs, and it’s just a static site you can clone and run yourself. The goal is simple—make browsing intentional rather than addictive. No logins, no ads, no algorithmic feeds; your data stays on your device.

Source on GitHub — clone it and make your own. To report a bug or suggest a change, open an issue or submit a pull request.
